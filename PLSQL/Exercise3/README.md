# Exercise 3: Stored Procedures

## Scenarios
- **Scenario 1**: Stored procedure `ProcessMonthlyInterest` updates the balance of savings accounts by applying a 1% monthly interest.
- **Scenario 2**: Stored procedure `UpdateEmployeeBonus` applies a performance-based bonus percentage to the salaries of employees in a given department.
- **Scenario 3**: Stored procedure `TransferFunds` transfers a specified amount between two accounts after verifying sufficient balance in the source account.
