# Exercise 1: Configuring a Basic Spring Application

## Scenario
Create a basic Spring Framework backend for a library management system.

## Steps
1. Create a Maven project `LibraryManagement` with Spring context dependencies.
2. Create XML configuration `applicationContext.xml` and define beans for `BookService` and `BookRepository`.
3. Implement `BookService` and `BookRepository` classes.
4. Load the ApplicationContext in `MainApplication` to test context load.
