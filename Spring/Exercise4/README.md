# Exercise 4: Creating and Configuring a Maven Project

## Scenario
Set up a new Maven project for the library management application and add Spring dependencies.

## Steps
1. Create a Maven project `LibraryManagement`.
2. Add dependencies for:
   - `spring-context`
   - `spring-aop`
   - `spring-webmvc`
3. Configure `maven-compiler-plugin` for Java 1.8 in `pom.xml`.
